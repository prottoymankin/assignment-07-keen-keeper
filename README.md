# Keen Kneeper
___

## 📝 Description
This Project demonstrate practical implementation of real world features like dynamic data handling, filtering, sorting, and data visualization in a modern fronted application.
___

## 🛠️ Technologies Used
- JavaScript (ES6+)
- React.js
- React Router DOM
- Tailwind CSS 
- Daisy Ui (Component Library)
- React-Toastify (Toast Notification Library)
- React Icons (Icon Library)
- Recharts
___

## 🚀 Core Features
- Dynamic Friends List fetching and rendering
- Multipage navigation using React Router
- Dynamic Timeline display
- Search timelines by friend names
- Filtering timelines by type (Call/Text/Video)
- Sorting timelines by Oldest -> Newest, Newest -> Oldest
- Interative Pie Chart for timeline statistics
